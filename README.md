# Sales & Inventory Analysis

This project analyzes sales and inventory data to generate business insights and support decision-making.

## Features
- Revenue calculation
- Product performance analysis
- Inventory monitoring (low stock alerts)
- Monthly sales tracking
- Automated report generation

## Tech Stack
- Python (pandas)

## Project Structure
- Modular architecture (data loading, processing, metrics, reporting)

## Use Case
Simulates a real-world business scenario where sales and inventory data are analyzed to optimize operations and support decision-making.

## How to Run
pip install -r requirements.txt  
python main.py
